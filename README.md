# rstudio-dmp
Docker for my custom rstudio environment

`v0.1.2` is the first working and tagged version.

Includes the following major R packages (and their dependencies):

* tidyverse
* lme4
* car
* psych
* lavaan
* fitdistrplus
* semPlot
* simsem 
* brms
* ggmcmc
* bnlearn
* viridis
* corrplot
* ggridges
* BDgraph
